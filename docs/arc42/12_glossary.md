# 12. Glossary

- **PlantUML:** A textual DSL used to describe UML diagrams; this library produces PlantUML text for diagrams.
- **UML:** Unified Modeling Language diagrams expressed in PlantUML syntax and generated by this libraryâ€™s methods.
- **StringBuilder extensions:** Extension methods on [`StringBuilder`][stringbuilder-docs] that emit PlantUML commands, such as [`UmlDiagramStart`][uml-diagram-start-source].
- **Diagram direction:** A typed enum (`DiagramDirection`) that encodes layout directions like left-to-right or top-to-bottom.
